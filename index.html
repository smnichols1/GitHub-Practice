Lesson 10 - Learning GitHub

Skylar Nichols March 21, 2024

In this week's lesson, we will be using Github and continue working on our Sprout & Grow project. In Visual Studio Code, include the following in your html document.

  • Use Emmett Abbreviations (shift +!) to give your document the basic HTML structure.
  • Title your page Lesson 10
  • Create an h1 heading "Lesson 10 - Learning GitHub,"
  • Create an h2 heading with your name and the current date
  • Be sure to include all of the HTML5 required elements such as (doctype - html - head - title - body).
  • Name your home page index.html.
